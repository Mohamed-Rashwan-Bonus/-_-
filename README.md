A modern, Neumorphic redesign of a university student portal ("UniHub") initially developed for East Port Said National University (EPNU). The project transforms a legacy Bootstrap interface into a sleek, customized Neumorphic UI focusing on usability, accessibility, and high visual impact.

## 🌟 Key Features
- **Neumorphism Design System:** Custom-built components mimicking soft UI extrusion to provide an elegant, premium look.
- **Fully Responsive:** Adapts seamlessly across desktop, tablet, and mobile devices.
- **RTL Built-in:** Designed purely from the ground up for Right-To-Left (RTL) Arabic text logic flow.
- **Interactive Sidebar & Navigation:** Mini-sidebar on desktop for maximum content width, expanding gracefully with pure CSS/JS overlay mechanics on mobile grids.
- **Dark Mode Support:** Foundation built for instantaneous toggling between light and dark themes (CSS variables mapped).

## 🚀 Pages Included
1. `home.html` — The main student dashboard with quick stats and overview.
2. `profile.html` — Detailed student information grid with modal integrations.
3. `registration.html` & `print-registration.html` — Course registration workflows.
4. `coursework.html` & `grades.html` — Grade checks and detailed semester trackers.
5. `military.html` & `military-results.html` — Dedicated reservation systems.
6. `login.html` — Beautifully engaging sign-in UI.
7. *More functional layout mocks included (attendance, absences, books).*

## 🛠️ Tech Stack
- **HTML5:** Semantic architecture.
- **CSS3:** Vanilla Custom properties, Flexbox/Grid systems, avoiding heavy framework bloat.
- **Vanilla JavaScript:** Micro-interactions, modal toggles, dynamic sidebars without requiring heavy front-end packages.

## 💻 How To Run
Clone the repository and open `login.html` or `home.html` in any web browser! No local server or build tools required.

```bash
git clone https://github.com/yourusername/unihub-redesign.git
cd unihub-redesign
```
